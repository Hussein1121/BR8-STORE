# BR8 STORE - متجر جراند الحياة الواقعية

**BR8 STORE** هو متجر مخصص لسيرفر **Grand RP** (النسخة الواقعية من لعبة Grand Theft Auto V). يهدف المشروع إلى توفير واجهة متجر شاملة للمنتجات داخل اللعبة، مع دعم كامل لتسجيل الدخول عبر **Discord OAuth2**.

## الهيكل الأساسي للمشروع

المشروع يحتوي على 9 صفحات رئيسية وكل صفحة تحتوي على وظائف محددة. يمكن تقسيم المشروع إلى المكونات التالية:

1. **الرئيسية (index.html)**:
   - تحتوي على واجهة المتجر مع عرض المنتجات الرئيسية.
   - يوجد رابط إلى صفحة المنتجات وصفحة الاتصال.

2. **المنتجات (products.html)**:
   - صفحة تعرض جميع المنتجات المتاحة في المتجر على شكل كروت (Cards).
   - يمكن للمستخدم الضغط على أي منتج للانتقال إلى صفحة تفاصيل المنتج.

3. **تفاصيل المنتج (product-details.html)**:
   - تعرض تفاصيل كل منتج.
   - تحتوي على زر "أضف إلى السلة".

4. **عربة التسوق (cart.html)**:
   - تعرض المنتجات التي أضافها المستخدم إلى سلة التسوق.
   - يحتوي على زر للدفع وانتقال إلى صفحة الدفع.

5. **الدفع (checkout.html)**:
   - صفحة لإتمام عملية الدفع.
   - تعرض تفاصيل الطلب والمجموع الكلي.

6. **حسابي (account.html)**:
   - تعرض بيانات المستخدم (بعد تسجيل الدخول عبر Discord).
   - يمكن للمستخدم تعديل بياناته الشخصية أو إتمام عمليات أخرى.

7. **تسجيل الدخول (login.html)**:
   - صفحة تسجيل الدخول عبر **Discord OAuth2**.
   - يتطلب الاتصال بـ Discord عبر الرابط المخصص للتسجيل.

8. **اتصل بنا (contact.html)**:
   - صفحة تحتوي على نموذج للتواصل مع إدارة المتجر.

9. **الدعم (support.html)**:
   - صفحة تحتوي على معلومات دعم فني.
